# Post-Production Checks

**Owner:** Shiva (`@shivakimothi-design`) · part of `/evaluation/`

Post-production = checks that run on the **finished, rendered asset** (not the concept text),
before spend. Both formats put a **G1/G2 gate first**, so passing post-prod means the gate passed.

| Format | Spec | Status |
|---|---|---|
| **Static** (FB performance image) | [`static-v1.md`](static-v1.md) | gate + Section-A disqualifiers + **6-criterion score (scoring v2.0)** |
| **Video** | post-prod v1.0 in [`../agents/README.md`](../agents/README.md) | v1.0 (multimodal re-run + craft hygiene) |

Source framework: [`Wiom_Creative_Evaluation_Framework_v2.0.docx`](Wiom_Creative_Evaluation_Framework_v2.0.docx)
(v2.0). The v1 framework is kept as a log in [`_archive/`](_archive/).
