# BFC creative learnings — pattern-tested (window Jun 30 – Jul 13, 2026)

*Confirmed by Shiva, 16 Jul 2026. Source: BFC-VOLUME campaign (goal: app
installs), Meta only, Delhi + Bharat judged separately, rankings on
same-campaign + same-ad-set pools. Method + full evidence tables:
`evaluation/creative-data-analyst/` (runs/2026-07-13_14d). Numbers pulled
16 Jul — installs keep maturing ~14 days; every learning below survived a
re-pull. A learning here = a pattern that repeated across the pool, never
a single-pair observation.*

## Learning 1 — Say "घर", show the router. Clicks stay flat; installs jump. *(confirmed, both geos)*

- "Recharge"/"net" language without a home anchor collects mobile-data
  clicks that die on the Play Store page.
- Clean proof: T-082/T-083/T-085 are the SAME black poster, only headline
  differs. CVR staircase as the home signal strengthens — Delhi: no घर
  6.1–7.2 → घर word (T-085) 9.3 → घर + router + comparison (T-063/064)
  10.8–11.7. Bharat independently: 10.7 → 13.5 → 20.5–24.7. CTR flat
  (0.51–0.58) throughout — the anchor costs zero clicks.
- Day-option chips are on ALL the black posters → day options alone don't
  convert; the anchor is the separating element.
- **Apply (install-lens scope):** every install ad carries घर + the product. "Home-ness" check
  = step one of any new static. *Scope caveat — this is an install/attention-CVR learning; it does
  **not** override booking outcomes. T-083 sits bottom of the CVR table yet is a top booking
  workhorse (₹1,202 CPBL, 91 bookings in its window) — so treat "say घर" as a craft default, not a
  kill/scale rule; booking decisions stay with the CPBL rule pass.*
- Sub-learning (single-variable, both geos): on identical bodies, the
  mechanism-statement headline ("जितनी ज़रूरत, उतना रिचार्ज") out-converts
  the challenge-question ("महीने का बंधन क्यों?") — 11.7 vs 10.8 Delhi,
  24.7 vs 20.5 Bharat — while the question pulls slightly more clicks.

## Learning 2 — Bharat: known local face + city named in line 1 ≈ 2× stop rate. *(strong; one gap)*

- Same ad set, same campaign: the four viewable creator films (Agra,
  Bareilly ×2, Moradabad — city named in the first line, local landmark
  shown) hook 22–25% of viewers; the AI-actor/no-place film (C-076) hooks
  11.2%.
- Caveats logged: creator fame and city words arrive together (not yet
  separable — the fame itself stops thumbs); only one non-creator film in
  the pool; H-006 (creator film, hook 17.8) not viewable.
- **Apply:** brief Bharat films through local creators, first line = the
  city. Gap-closer test: same script, city callout, unknown face.

## Learning 3 — Open the film inside the home-net world, not the mobile-data world. *(directional, n=3)*

- Delhi videos: home-net entries convert better — C-076 ("नेट लगवाने का
  कबसे सोच रहा था") CVR 23.4, C-069 ("घर का net चाहिए? महीने का plan भारी?")
  16.4, vs C-077's mobile-data story entry ("फोन से चलता था… शाम तक ख़त्म")
  13.7 — despite C-077 winning attention (best CTR).
- Same logic as Learning 1 across formats: whoever you attract is who you
  convert. Only 3 films → directional, not confirmed.
- **Apply:** story/emotion welcome, but the first seconds place the viewer
  in the HOME internet situation. Test: re-cut C-077 with a home-first
  opening line.

## Checked and parked

- **Everyday-examples explanation (doodh-dahi/newspaper) → watching**:
  only ONE viewable film uses the device (C-069); Delhi's top holds are
  tied (C-077 23.1 / C-069 22.9). No pattern possible — parked per the
  rule "if nobody else uses it, ignore."
- **Empathy hook (own problem vs advice)**: dropped — the big hook lifts
  always co-occur with face + place, so the trait isn't isolated.

## Side notes

- **Targeting (for the media side, not creative):** the same film converts
  clicks ~half as well in city-targeted ad sets vs the shared broad ad set
  — 6 of 6 films. Creative rankings must never mix ad sets.
- **H-006**: on clean numbers it is Bharat's best converter (39.7) and
  best holder (28.7) with the weakest creator hook — the C-076 personality
  in creator form. File unavailable; top content read if found.
- **C-076 production fix**: on-screen typo "montly plan" in the film's key
  line — correct before further spend.
