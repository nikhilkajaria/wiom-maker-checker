# Explore — open experiment sandbox

**Owner:** open posting (no required reviewer) · **Promotion governed by:** Shiva

Anyone may drop a messaging experiment here — no owner approval needed to post.

**Promotion rule:** moving an experiment *into the thesis* edits `/messaging/thesis/`, which
**is** gated (Shiva approves). So you can experiment freely here; it only changes the
official thesis once Shiva promotes it.
