# L1 Creative — the "From → To" Element Library

*The building blocks that carry an L1 user from the familiar WiFi frame to "recharge-wala ghar ka net."*

**The journey.** L1 is the ready buyer already shopping "WiFi." The job is **reclassification**: start inside the WiFi / home-internet frame they already have (**FROM**), then land them on **recharge-wala ghar ka net + flexibility** (**TO**). Capture in familiarity, convert in the category.

**Columns.** *Tested?* = did we run it. *Did it work?* = the result from the learnings log — ✅ Yes · 🟠 Mixed · 🔴 No · — not yet.

### Rules (guardrails)
- Use "WiFi" only as the **on-ramp**, never as the destination / Wiom's identity. Destination noun = **"net" / "ghar ka net"**, never "recharge" alone (reads as mobile).
- No competitor names. Don't lead with speed / Mbps. Don't promise "cheaper." Clean L1: no L2 monthly-waste or L3 hotspot/top-up hooks.

## BRIDGE — flip FROM → TO in one move
*Devices that take the WiFi frame and hand it to the category. These do the heavy lifting on attention.*

| Element | Type | How it works | Tested? | Did it work? |
|---|---|---|---|---|
| Crossed-WiFi → Wiom ("ये WiFi नहीं, Wiom है") | Combo | Takes the WiFi frame, negates it, hands off to ghar ka net | Yes | 🟠 Mixed — best hook, needs the carousel's 2nd card to convert |
| Split-screen comparison (need vs no-need) | Combo | Contrast pulls attention, then lands the category | Yes | ✅ Yes — strong (0.65% CTR, #3 booking) |
| Question → answer ("घर का नेट चाहिए? अब रिचार्ज पर") | Word | Capture the need, resolve to the category | Yes | ✅ Yes |
| "Mobile recharge ✗ → घर का नेट ✓" contrast | Combo | Re-anchors "recharge" away from mobile | Not yet · briefed | — |

## How to use this
- Every L1 creative = **one FROM element + one TO element, joined by a BRIDGE.**
- As each test resolves, fill in *Tested?* and *Did it work?*. Promote what works; retire what fails comprehension (e.g. the device montage → OTT misread).
